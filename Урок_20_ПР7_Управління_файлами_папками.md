---

## Мета роботи

Набути практичних навичок створення, копіювання, переміщення, перейменування та видалення файлів і папок у Windows за допомогою Провідника (File Explorer) та командного рядка (CMD).

---

## Необхідне програмне забезпечення

- Windows 10 або Windows 11
- Провідник (File Explorer)
- Командний рядок (cmd.exe)
- Текстовий редактор «Блокнот» (Notepad)

---

## Короткий теоретичний вступ

### Файли та каталоги

**Файл** — іменована область даних на диску, що має ім'я та розширення. Наприклад: `звіт.docx`, `фото.jpg`, `програма.exe`.

**Каталог (папка)** — контейнер для зберігання файлів та інших каталогів. Каталоги утворюють **ієрархічну (деревоподібну) структуру**.

### Шлях до файлу

**Абсолютний шлях** — повний шлях від кореня диска:
```
C:\Users\Student\Documents\Звіт\звіт.txt
```

**Відносний шлях** — шлях відносно поточного каталогу:
```
Documents\Звіт\звіт.txt
```

### Складові імені файлу

| Елемент | Приклад | Пояснення |
|---------|---------|-----------|
| Ім'я файлу | `звіт` | Основна частина назви |
| Розширення | `.txt` | Визначає тип файлу |
| Повне ім'я | `звіт.txt` | Ім'я + розширення |

### Основні розширення файлів

| Розширення | Тип файлу |
|-----------|-----------|
| `.txt` | Текстовий файл |
| `.docx` | Документ Microsoft Word |
| `.xlsx` | Таблиця Microsoft Excel |
| `.html` | Веб-сторінка |
| `.jpg`, `.png` | Зображення |
| `.exe` | Виконуваний файл (програма) |

### Символи-маски (шаблони)

| Символ | Значення | Приклад |
|--------|----------|---------|
| `*` | Будь-яка кількість будь-яких символів | `*.txt` — всі файли з розширенням .txt |
| `?` | Рівно один будь-який символ | `??.txt` — файли з ім'ям із двох символів |

## Таблиця основних CMD-команд для роботи з файлами та папками

| Команда | Призначення | Приклад |
|---------|-------------|---------|
| `mkdir` (або `md`) | Створити папку | `mkdir C:\Test\Docs` |
| `rmdir` (або `rd`) | Видалити порожню папку | `rmdir C:\Test\Docs` |
| `rmdir /s /q` | Видалити папку з вмістом (без підтвердження) | `rmdir /s /q C:\Test` |
| `copy` | Копіювати файл | `copy file.txt D:\Backup\` |
| `xcopy` | Копіювати файли та папки з підпапками | `xcopy C:\Docs D:\Backup /s /e` |
| `robocopy` | Надійне копіювання (підтримує повтори, потоки) | `robocopy C:\Docs D:\Backup /e` |
| `move` | Перемістити файл або папку | `move file.txt D:\Archive\` |
| `del` | Видалити файл | `del file.txt` |
| `ren` | Перейменувати файл або папку | `ren old.txt new.txt` |
| `dir` | Показати вміст каталогу | `dir C:\Test` |
| `dir /s` | Показати вміст із підпапками | `dir C:\Test /s` |
| `dir /a:h` | Показати приховані файли | `dir /a:h` |
| `attrib` | Переглянути/змінити атрибути файлу | `attrib +h file.txt` |
| `cd` | Змінити поточний каталог | `cd C:\Test\Docs` |
| `tree` | Показати дерево каталогів | `tree C:\Test` |
| `type` | Показати вміст текстового файлу | `type file.txt` |
| `echo` | Створити файл із текстом | `echo Hello > file.txt` |

### Завдання 1. Створення ієрархії папок через Провідник

**Мета:** навчитися створювати складну структуру папок через графічний інтерфейс.

Створіть таку ієрархію папок через Провідник Windows:

```
C:\ПР7_Прізвище\
├── Документи\
│   ├── Звіти\
│   │   ├── 2024\
│   │   └── 2025\
│   └── Листи\
├── Медіа\
│   ├── Фото\
│   │   ├── Відпустка\
│   │   └── Навчання\
│   ├── Відео\
│   └── Музика\

    ├── Встановлені\
    └── Портативні\
```

**Покрокова інструкція (GUI-спосіб):**

1. Відкрийте Провідник (натисніть `Win + E`).
2. Перейдіть до диска `C:\`.
3. Створіть нову папку: натисніть `Ctrl + Shift + N` або клацніть правою кнопкою миші → **Створити** → **Папку**.

7. Відкрийте папку `Документи` та створіть у ній: `Звіти`, `Листи`.
8. Відкрийте папку `Звіти` та створіть: `2024`, `2025`.
9. Поверніться назад (натисніть `Alt + ←`) і повторіть для папок `Медіа` та `Програми` згідно зі схемою.

**Перевірте результат:** у Провіднику натисніть на адресному рядку та введіть шлях `C:\ПР7_Прізвище`, переконайтеся, що всі папки створені.

Створіть **ту саму ієрархію** (як у Завданні 1), але в папці `C:\ПР7_CMD_Прізвище` за допомогою команди `mkdir`.

**Покрокова інструкція (CMD-спосіб):**

1. Відкрийте командний рядок: натисніть `Win + R`, введіть `cmd`, натисніть Enter.

```cmd

mkdir C:\ПР7_CMD_Прізвище\Документи\Звіти\2025
mkdir C:\ПР7_CMD_Прізвище\Документи\Листи

mkdir C:\ПР7_CMD_Прізвище\Медіа
mkdir C:\ПР7_CMD_Прізвище\Медіа\Фото
mkdir C:\ПР7_CMD_Прізвище\Медіа\Фото\Відпустка
mkdir C:\ПР7_CMD_Прізвище\Медіа\Фото\Навчання
```

3. Перевірте створену структуру командою:
```cmd

```

> **Зробіть знімок екрана** результату виконання команди `tree`.

### Завдання 3. Створення файлів різних типів

**Мета:** навчитися створювати файли різних форматів та розміщувати їх у відповідних папках.

4. У тій самій папці створіть ще один текстовий документ `звіт_лютий.txt` із текстом: `Це звіт за лютий 2025 року.`

```cmd
cd C:\ПР7_CMD_Прізвище\Документи\Звіти\2025
```

> **Примітка:** У CMD символи `<` та `>` є спеціальними (перенаправлення), тому їх треба екранувати символом `^`.

### Завдання 4. Копіювання файлів

**Мета:** навчитися копіювати файли через Провідник та CMD різними способами.

#### GUI-спосіб (Провідник):

1. Відкрийте папку `C:\ПР7_Прізвище\Документи\Звіти\2025`.
2. Виділіть файл `звіт_січень.txt`.
3. Натисніть `Ctrl + C` (копіювати).
4. Перейдіть до папки `C:\ПР7_Прізвище\Документи\Звіти\2024`.
5. Натисніть `Ctrl + V` (вставити).
6. Файл скопійовано!

> **Альтернативний GUI-спосіб:** клацніть правою кнопкою миші на файлі → **Копіювати**, потім у цільовій папці клацніть правою кнопкою → **Вставити**.

#### CMD-спосіб — команда `copy`:

```cmd
copy C:\ПР7_CMD_Прізвище\Документи\Звіти\2025\звіт_січень.txt C:\ПР7_CMD_Прізвище\Документи\Звіти\2024\
```

**Результат:** файл `звіт_січень.txt` скопійовано з папки `2025` до папки `2024`.

#### CMD-спосіб — команда `xcopy` (копіювання з підпапками):

```cmd
xcopy C:\ПР7_CMD_Прізвище\Документи C:\ПР7_CMD_Прізвище\Медіа\Копія_Документів\ /s /e /y
```

| Ключ | Значення |
|------|----------|
| `/s` | Копіювати підпапки (крім порожніх) |
| `/e` | Копіювати всі підпапки, включно з порожніми |
| `/y` | Не запитувати підтвердження при перезаписі |

#### CMD-спосіб — команда `robocopy` (надійне копіювання):

```cmd
robocopy C:\ПР7_CMD_Прізвище\Документи\Звіти C:\ПР7_CMD_Прізвище\Медіа\Резервна_копія /e
```

| Ключ | Значення |
|------|----------|
| `/e` | Копіювати всі підпапки, включно з порожніми |
| `/mir` | Дзеркальне копіювання (синхронізація) |
| `/z` | Режим відновлення при перерві |
| `/mt:8` | Багатопотокове копіювання (8 потоків) |

> **Зробіть знімок екрана** — результат копіювання через Провідник та результат виконання `robocopy`.

### Завдання 7. Видалення файлів та папок

**Мета:** навчитися видаляти файли різними способами — у Кошик та назавжди.

#### GUI-спосіб (Провідник):

**Видалення у Кошик:**
1. Відкрийте папку `C:\ПР7_Прізвище\Документи\Звіти\2024`.
2. Виділіть файл `звіт_січень.txt`.
3. Натисніть клавішу `Delete`.
4. Файл переміщено до Кошика. Відкрийте Кошик на робочому столі та переконайтеся, що файл там.

**Відновлення з Кошика:**
1. У Кошику клацніть правою кнопкою на файлі → **Відновити**.
2. Файл повернеться у папку, з якої був видалений.

**Видалення назавжди (без Кошика):**
1. Виділіть файл `звіт_січень.txt` у папці `2024`.
2. Натисніть `Shift + Delete`.
3. Підтвердіть видалення у діалоговому вікні.
4. Файл видалено **назавжди** — його немає в Кошику!

> **Увага!** Файли, видалені через `Shift + Delete`, **неможливо відновити** стандартними засобами Windows!

#### CMD-спосіб:

**Видалення файлу:**
```cmd
del C:\ПР7_CMD_Прізвище\Документи\Звіти\2024\звіт_січень.txt
```

> **Примітка:** Команда `del` видаляє файли **назавжди** (не в Кошик)!

**Видалення всіх файлів .txt у папці:**
```cmd
del C:\ПР7_CMD_Прізвище\Документи\Звіти\2024\*.txt
```

**Видалення порожньої папки:**
```cmd
rmdir C:\ПР7_CMD_Прізвище\Документи\Звіти\2024
```

**Видалення папки з усім вмістом (файли + підпапки):**
```cmd
rmdir /s /q C:\ПР7_CMD_Прізвище\Документи\Звіти\2024
```

| Ключ | Значення |
|------|----------|
| `/s` | Видалити підпапки та файли всередині |
| `/q` | Без запиту підтвердження (quiet mode) |

> **Зробіть знімок екрана** — Кошик із видаленим файлом та результат команди `del`.

### Завдання 8. Пошук файлів за маскою (шаблоном)

**Мета:** навчитися шукати файли за допомогою символів-масок `*` та `?`.

#### GUI-спосіб (Провідник):

1. Відкрийте Провідник і перейдіть до папки `C:\ПР7_Прізвище`.
2. У полі пошуку (верхній правий кут) введіть: `*.txt`
3. Провідник покаже всі файли з розширенням `.txt` у цій папці та підпапках.
4. Спробуйте інші маски пошуку:
   - `Звіт*` — усі файли, що починаються зі слова «Звіт»
   - `*.html` — усі HTML-файли
   - `??.*` — файли, ім'я яких складається з двох символів (будь-яке розширення)

#### CMD-спосіб — команда `dir` з масками:

Перейдіть до робочої папки:
```cmd
cd C:\ПР7_CMD_Прізвище
```

**Пошук усіх .txt файлів у поточній папці та підпапках:**
```cmd
dir *.txt /s
```

**Пошук файлів з ім'ям із двох символів:**
```cmd
dir ??.* /s
```

**Пошук усіх файлів, що починаються на «Звіт»:**
```cmd
dir Звіт*.* /s
```

**Пошук усіх HTML-файлів:**
```cmd
dir *.html /s
```

**Пояснення ключів:**

| Ключ | Значення |
|------|----------|
| `/s` | Шукати у підпапках (рекурсивно) |
| `/b` | Короткий формат (лише імена файлів) |
| `/p` | Посторінковий вивід |

**Приклад розширеного пошуку:**
```cmd
dir *.txt /s /b
```
Ця команда покаже повні шляхи до всіх файлів `.txt` у папці та її підпапках.

> **Зробіть знімок екрана** з результатами пошуку через Провідник та CMD.

### Завдання 9. Перегляд та зміна атрибутів файлів

**Мета:** навчитися переглядати та змінювати атрибути файлів (тільки для читання, прихований, архівний, системний).

#### Теоретична довідка — атрибути файлів

| Атрибут | Літера | Значення |
|---------|--------|----------|
| Read-only (Тільки для читання) | `R` | Файл захищений від змін |
| Hidden (Прихований) | `H` | Файл не відображається у Провіднику |
| System (Системний) | `S` | Файл є системним |
| Archive (Архівний) | `A` | Файл змінено після останнього резервного копіювання |

#### GUI-спосіб (Провідник):

1. Відкрийте папку `C:\ПР7_Прізвище\Документи\Листи`.
2. Клацніть правою кнопкою миші на файлі `лист_директору.txt` → **Властивості**.
3. У нижній частині вікна побачите прапорці:
   - **Тільки читання** (Read-only)
   - **Прихований** (Hidden)
4. Поставте прапорець **Тільки читання** → натисніть **OK**.
5. Спробуйте відкрити файл і змінити текст — збереження має бути заблоковане.
6. Поверніться до Властивостей та зніміть прапорець **Тільки читання**.

#### CMD-спосіб — команда `attrib`:

**Перегляд атрибутів файлу:**
```cmd
attrib C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

Результат буде виглядати приблизно так:
```
A            C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```
Літера `A` означає, що встановлений атрибут «Архівний».

**Встановлення атрибуту «Тільки для читання»:**
```cmd
attrib +r C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

**Перевірка:**
```cmd
attrib C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```
Тепер результат має показати `A    R` — файл одночасно архівний та тільки для читання.

**Зняття атрибуту «Тільки для читання»:**
```cmd
attrib -r C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

**Перегляд атрибутів усіх файлів у папці:**
```cmd
attrib C:\ПР7_CMD_Прізвище\Документи\Листи\*.*
```

> **Зробіть знімок екрана** — Властивості файлу з атрибутами та результат команди `attrib`.

### Завдання 10. Робота з прихованими файлами

**Мета:** навчитися приховувати файли та показувати їх у Провіднику та через CMD.

#### Крок 1. Приховати файл через CMD

```cmd
attrib +h C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

Перевірте у Провіднику: відкрийте папку `C:\ПР7_CMD_Прізвище\Документи\Листи` — файл `лист_директору.txt` **зник**!

#### Крок 2. Показати приховані файли у Провіднику

**Для Windows 11:**
1. У Провіднику натисніть **Вид** (View) у верхній панелі.
2. Наведіть курсор на **Показати** (Show).
3. Поставте прапорець **Приховані елементи** (Hidden items).
4. Тепер прихований файл з'явиться з напівпрозорою іконкою.

**Для Windows 10:**
1. У Провіднику перейдіть на вкладку **Вид** (View).
2. Поставте прапорець **Приховані елементи** (Hidden items).

**Через Параметри папок (обидві версії Windows):**
1. У Провіднику натисніть на три крапки `...` → **Параметри** (Options).
2. Перейдіть на вкладку **Вид** (View).
3. У списку **Додаткові параметри** знайдіть пункт **Приховані файли та папки**.
4. Оберіть **Показувати приховані файли, папки та диски**.
5. Натисніть **OK**.

#### Крок 3. Побачити приховані файли через CMD

```cmd
dir /a:h C:\ПР7_CMD_Прізвище\Документи\Листи
```

Ключ `/a:h` показує лише приховані файли.

#### Крок 4. Зняти атрибут «прихований»

```cmd
attrib -h C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

Тепер файл знову видимий для всіх.

#### Крок 5 (додатково). Встановити атрибути «прихований» + «системний»

Файл із обома атрибутами ще важче виявити:
```cmd
attrib +h +s C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

Такий файл не з'явиться навіть при ввімкненому показі прихованих файлів — потрібно додатково зняти прапорець **Приховувати захищені системні файли** в параметрах папок.

**Зняття обох атрибутів:**
```cmd
attrib -h -s C:\ПР7_CMD_Прізвище\Документи\Листи\лист_директору.txt
```

> **Зробіть знімок екрана** — папка з прихованим файлом (до та після ввімкнення показу) та результат `dir /a:h`.
