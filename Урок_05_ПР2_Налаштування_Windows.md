## 1. Мета роботи

- Навчитися налаштовувати робоче середовище операційної системи Windows 10/11 відповідно до власних потреб.
- Оволодіти навичками персоналізації інтерфейсу, керування програмами автозапуску та використання гарячих клавіш.
- Зрозуміти різницю між додатком "Параметри" та "Панеллю керування".

---

## 2. Теоретичний вступ

### 2.1. Параметри (Settings) та Панель керування (Control Panel)

Операційна система Windows має два основних інструменти для налаштування системи:

**Параметри (Settings)** — сучасний додаток з графічним інтерфейсом, який з'явився у Windows 8 і став основним засобом налаштування у Windows 10/11. Має зручний, інтуїтивно зрозумілий дизайн з великими іконками та кольоровими елементами. Добре адаптований для сенсорних екранів. Відкривається комбінацією клавіш **Win + I** або через меню "Пуск".

**Панель керування (Control Panel)** — класичний інструмент налаштування, що існує з перших версій Windows. Містить розширені параметри системи, які ще не повністю перенесені до "Параметрів". Відкривається через пошук Windows (ввести "Панель керування" або "Control Panel").

### 2.2. Різниця між Параметрами та Панеллю керування

| Характеристика | Параметри (Settings) | Панель керування (Control Panel) |
|---|---|---|
| **Інтерфейс** | Сучасний, адаптивний, зручний для сенсорних екранів | Класичний, детальний, розрахований на мишу і клавіатуру |
| **Технологія** | Побудований на сучасних технологіях (UWP/WinUI) | Побудований на класичних технологіях (Win32) |
| **Основне призначення** | Повсякденні налаштування: персоналізація, мережа, облікові записи, оновлення | Розширені налаштування: BitLocker, індексація файлів, просунуте керування пристроями |
| **Перспективи** | Microsoft поступово переносить усі налаштування сюди | Поступово втрачає функції, але досі необхідний для деяких задач |
| **Виклик** | Win + I | Пошук Windows або команда `control` |

> **Зверніть увагу:** Microsoft поступово переносить усі налаштування з Панелі керування до Параметрів. У деяких випадках, коли ви відкриваєте елемент у Панелі керування, система автоматично перенаправляє вас до відповідного розділу Параметрів. Наразі обидва інструменти співіснують.

## 3. Практичні завдання

### 3.1. Завдання 1. Персоналізація робочого столу

**Мета:** Навчитися змінювати шпалери, тему та кольорову схему Windows.

#### Крок 1. Зміна шпалер (фонового зображення)

1. Натисніть **правою кнопкою миші** на порожньому місці робочого столу.
2. Оберіть пункт **"Персоналізація"** (або перейдіть: Параметри → Персоналізація → Фон).
3. У розділі **"Персоналізація фону"** оберіть один із варіантів:
   - **Зображення** — встановити конкретне фото або картинку.
   - **Суцільний колір** — однотонний фон обраного кольору.
   - **Слайд-шоу** — автоматична зміна зображень з обраної папки.
   - **Windows: цікаве** (Windows Spotlight) — система автоматично завантажує нові шпалери щодня.
4. Якщо обрали "Зображення", натисніть **"Огляд фото"** та знайдіть потрібний файл на комп'ютері.
5. Оберіть **розташування зображення**: Заповнення, За розміром, Розтягнути, Замостити, По центру, Проміжок.

> **Порада:** Можна також натиснути правою кнопкою миші на будь-яке зображення у Провіднику файлів та обрати "Встановити як фон робочого столу".

#### Крок 2. Зміна теми

1. Перейдіть: **Параметри → Персоналізація → Теми**.
2. У розділі **"Поточна тема"** оберіть одну з попередньо встановлених тем:
   - Windows 11 має кілька стандартних тем: "Windows (світла)", "Windows (темна)", "Glow", "Sunrise", "Flow".
3. Для завантаження додаткових тем натисніть **"Переглянути теми"** — відкриється Microsoft Store з колекцією безкоштовних і платних тем (понад 400 варіантів).
4. Оберіть тему, натисніть **"Отримати"** або **"Інсталювати"**, потім **"Відкрити"** та застосуйте її.

#### Крок 3. Зміна кольорової схеми

1. Перейдіть: **Параметри → Персоналізація → Кольори**.
2. **Оберіть режим:**
   - **Світлий** — світлий інтерфейс (меню, вікна, панель задач).
   - **Темний** — темний інтерфейс (знижує навантаження на очі при роботі в темному приміщенні).
   - **Настроюваний** — дозволяє окремо обрати режим для Windows і для додатків.
3. У розділі **"Акцентний колір"** оберіть:
   - **Автоматично** — колір підбирається на основі шпалер.
   - **Вручну** — оберіть колір зі спектру кольорів.
4. Увімкніть перемикач **"Показувати акцентний колір на кнопці Пуск і панелі задач"** для більш цілісного вигляду.
5. Увімкніть або вимкніть **"Ефекти прозорості"** для панелі задач та вікон.

### 3.2. Завдання 2. Налаштування панелі задач

**Мета:** Навчитися налаштовувати вигляд і поведінку панелі задач.

#### Крок 1. Відкриття налаштувань панелі задач

1. Натисніть **правою кнопкою миші** на порожньому місці панелі задач.
2. Оберіть **"Параметри панелі задач"** (або: Параметри → Персоналізація → Панель задач).

#### Крок 2. Налаштування елементів панелі задач

У розділі **"Елементи панелі задач"** увімкніть або вимкніть:
- **Пошук** — оберіть варіант відображення: приховати, лише значок, значок із підписом, або поле пошуку.
- **Подання задач (Task View)** — кнопка для перегляду відкритих вікон та віртуальних робочих столів.
- **Міні-додатки (Widgets)** — панель з погодою, новинами та календарем.
- **Чат** — швидкий доступ до Microsoft Teams.

#### Крок 3. Закріплення та відкріплення програм

- **Закріпити програму:** Відкрийте потрібну програму, натисніть правою кнопкою миші на її значку на панелі задач → **"Закріпити на панелі задач"**.
- **Відкріпити програму:** Натисніть правою кнопкою миші на закріпленому значку → **"Відкріпити від панелі задач"**.
- Перетягуйте значки на панелі задач для зміни їх порядку.

#### Крок 4. Зміна розташування значків

1. У налаштуваннях панелі задач знайдіть розділ **"Поведінка панелі задач"**.
2. У пункті **"Вирівнювання панелі задач"** оберіть:
   - **По центру** — сучасний стиль Windows 11 (за замовчуванням).
   - **Ліворуч** — класичне розташування, як у попередніх версіях Windows.

#### Крок 5. Увімкнення автоприховування

1. У розділі **"Поведінка панелі задач"** увімкніть перемикач **"Автоматично приховувати панель задач"**.
2. Тепер панель задач ховатиметься, коли ви нею не користуєтеся, і з'являтиметься при наведенні курсору миші до нижнього краю екрана.

#### Крок 6. Налаштування системної області (system tray)

1. У розділі **"Піктограми системної області"** увімкніть або вимкніть відображення значків:
   - Меню пера
   - Сенсорна клавіатура
   - Віртуальний тачпад
2. У розділі **"Інші піктограми системної області"** керуйте видимістю значків фонових програм.

### 3.3. Завдання 3. Налаштування меню "Пуск"

**Мета:** Навчитися налаштовувати меню "Пуск" для швидкого доступу до потрібних програм.

#### Крок 1. Закріплення додатків у меню "Пуск"

1. Натисніть кнопку **"Пуск"** (або клавішу **Win**).
2. Натисніть **"Усі додатки"** у верхньому правому куті.
3. Знайдіть потрібну програму, натисніть на неї **правою кнопкою миші** → **"Закріпити на початковому екрані"**.
4. Програма з'явиться в розділі **"Закріплені"** меню "Пуск".

#### Крок 2. Відкріплення та переміщення додатків

- **Відкріпити:** Натисніть правою кнопкою миші на закріпленому додатку → **"Відкріпити від початкового екрана"**.
- **Перемістити:** Затисніть ліву кнопку миші на значку та перетягніть його на потрібне місце.

#### Крок 3. Налаштування макету меню "Пуск"

1. Перейдіть: **Параметри → Персоналізація → Пуск**.
2. Оберіть макет:
   - **Більше закріплень** — збільшує область для закріплених додатків.
   - **За замовчуванням** — стандартний баланс між закріпленими додатками та рекомендованими.
   - **Більше рекомендацій** — збільшує область рекомендованих файлів та додатків.
3. Увімкніть або вимкніть:
   - Показувати нещодавно додані програми.
   - Показувати найбільш використовувані програми.
   - Показувати нещодавно відкриті елементи у меню "Пуск", списках переходів та Провіднику файлів.

#### Крок 4. Додавання папок поряд із кнопкою живлення

1. У розділі **Параметри → Персоналізація → Пуск → Папки** увімкніть відображення швидких посилань:
   - Параметри
   - Провідник файлів
   - Документи
   - Завантаження
   - Зображення
   - Мережа
   - Особиста папка

### 3.4. Завдання 4. Налаштування дисплею

**Мета:** Навчитися змінювати роздільну здатність, масштабування та орієнтацію екрана.

#### Крок 1. Відкриття параметрів дисплею

1. Натисніть правою кнопкою миші на робочому столі → **"Параметри дисплею"**.
2. Або перейдіть: **Параметри → Система → Дисплей**.

#### Крок 2. Зміна роздільної здатності

1. Прокрутіть до розділу **"Роздільна здатність дисплею"**.
2. Натисніть на випадаючий список і оберіть потрібне значення.
3. Рекомендовано обирати значення з позначкою **(Рекомендовано)** — це відповідає рідній роздільній здатності вашого монітора.
4. Після зміни з'явиться запит на підтвердження. Натисніть **"Зберегти зміни"** протягом 15 секунд, інакше налаштування повернуться до попередніх.

> **Типові роздільні здатності:** 1920x1080 (Full HD), 2560x1440 (QHD), 3840x2160 (4K UHD), 1366x768 (для ноутбуків).

#### Крок 3. Зміна масштабування

1. У розділі **"Масштаб"** оберіть відсоток збільшення: 100%, 125%, 150%, 175%, 200%.
2. Масштабування збільшує розмір тексту, значків та елементів інтерфейсу **без зниження якості зображення**.
3. Для моніторів з високою роздільною здатністю (4K) зазвичай рекомендується 150% або 200%.

> **Увага:** Після зміни масштабування деякі програми можуть потребувати перезапуску або виходу з облікового запису для коректного відображення.

#### Крок 4. Зміна орієнтації

1. У розділі **"Орієнтація дисплею"** оберіть:
   - **Альбомна** — стандартне горизонтальне відображення.
   - **Книжкова** — вертикальне відображення (зручно для роботи з документами та кодом).
   - **Альбомна (перевернута)** — горизонтальне, повернуте на 180 градусів.
   - **Книжкова (перевернута)** — вертикальне, повернуте на 180 градусів.
2. Після зміни орієнтації також потрібно фізично повернути монітор, якщо це зовнішній дисплей.

### 3.5. Завдання 5. Налаштування дати, часу та мови

**Мета:** Навчитися встановлювати правильні дату, час, часовий пояс та мову інтерфейсу.

#### Крок 1. Налаштування дати та часу

1. Натисніть **правою кнопкою миші** на годиннику в правому нижньому куті панелі задач → **"Налаштування дати й часу"**.
2. Або перейдіть: **Параметри → Час та мова → Дата й час**.
3. Увімкніть **"Автоматичне встановлення часу"** — система синхронізуватиме час через інтернет.
4. Увімкніть **"Автоматичне встановлення часового поясу"** або виберіть вручну: **(UTC+02:00) Київ**.
5. Для ручного встановлення дати та часу: вимкніть автоматичне встановлення → натисніть **"Змінити"** → введіть правильні значення → **"Змінити"**.

#### Крок 2. Додавання та перемикання мов

1. Перейдіть: **Параметри → Час та мова → Мова та регіон**.
2. У розділі **"Бажані мови"** натисніть **"Додати мову"**.
3. Знайдіть потрібну мову (наприклад, English (United States)) та натисніть **"Далі"** → **"Інсталювати"**.
4. Для перемикання мов введення використовуйте комбінацію **Win + Пробіл** або натисніть на індикатор мови на панелі задач.

#### Крок 3. Зміна регіональних параметрів

1. У розділі **"Регіон"** оберіть вашу країну/регіон: **Україна**.
2. Це впливає на формат дати, часу, валюти та чисел.
3. Для детальних налаштувань формату натисніть **"Регіональний формат"** → **"Змінити формати"**.

---

### 3.6. Завдання 6. Налаштування звуку та сповіщень

**Мета:** Навчитися керувати рівнем гучності, звуковими пристроями та системними сповіщеннями.

#### Крок 1. Налаштування звуку

1. Натисніть **правою кнопкою миші** на значку динаміка в системній області → **"Параметри звуку"**.
2. Або перейдіть: **Параметри → Система → Звук**.
3. У розділі **"Вивід"** оберіть пристрій відтворення (динаміки, навушники).
4. Регулюйте **загальну гучність** за допомогою повзунка.
5. У розділі **"Введення"** оберіть мікрофон та налаштуйте його гучність.
6. Натисніть **"Мікшер гучності"** для керування гучністю окремих програм.

#### Крок 2. Налаштування системних звуків

1. Прокрутіть вниз до розділу **"Додатково"** → **"Інші параметри звуку"**.
2. У вікні, що відкриється, перейдіть на вкладку **"Звуки"**.
3. Тут можна:
   - Обрати **звукову схему** (Windows за замовчуванням, без звуків тощо).
   - Налаштувати звуки для окремих подій (сповіщення, помилки, запуск системи тощо).

#### Крок 3. Налаштування сповіщень

1. Перейдіть: **Параметри → Система → Сповіщення**.
2. Увімкніть або вимкніть **загальний перемикач сповіщень**.
3. Нижче налаштуйте сповіщення для кожної програми окремо: увімкніть або вимкніть, оберіть тип (банер, звук, центр сповіщень).
4. Увімкніть **"Не турбувати"** для тимчасового блокування сповіщень під час роботи.
5. Налаштуйте **"Фокусування"** — режим, що автоматично вимикає сповіщення за певних умов (під час гри, повноекранного режиму тощо).

### 3.7. Завдання 7. Керування автозапуском програм

**Мета:** Навчитися контролювати, які програми запускаються автоматично при увімкненні комп'ютера.

> **Навіщо це потрібно?** Багато програм після інсталяції автоматично додають себе в автозапуск. З часом таких програм накопичується багато, що значно сповільнює завантаження Windows та загальну швидкодію комп'ютера.

#### Спосіб 1. Через Диспетчер завдань (рекомендовано)

1. Натисніть комбінацію клавіш **Ctrl + Shift + Esc** для відкриття Диспетчера завдань.
2. У лівій панелі (або у вкладках зверху) перейдіть до розділу **"Автозавантаження"** (Startup apps).
3. Ви побачите список усіх програм автозапуску з інформацією:
   - **Ім'я програми** — назва додатку.
   - **Видавець** — розробник програми.
   - **Стан** — "Увімкнено" або "Вимкнено".
   - **Вплив на запуск** — оцінка впливу на швидкість завантаження:
     - **Низький** — програма майже не впливає на завантаження (менше 300 мс процесорного часу).
     - **Середній** — помірний вплив (300 мс — 1 с процесорного часу).
     - **Високий** — суттєвий вплив (більше 1 с процесорного часу або більше 3 МБ дискових операцій).
4. Для вимкнення програми: натисніть на ній **правою кнопкою миші** → **"Вимкнути"**.
5. Для увімкнення: натисніть правою кнопкою миші → **"Увімкнути"**.

> **Порада:** Спочатку вимикайте програми з **високим** впливом на запуск. Не вимикайте антивірусні програми та драйвери пристроїв!

#### Спосіб 2. Через Параметри Windows

1. Перейдіть: **Параметри → Додатки → Автозавантаження** (Win + I → Apps → Startup).
2. Перемикайте повзунки біля кожної програми для увімкнення або вимкнення автозапуску.

### 3.8. Завдання 8. Налаштування миші та клавіатури

**Мета:** Навчитися налаштовувати параметри миші та клавіатури для зручної роботи.

#### Крок 1. Налаштування миші

1. Перейдіть: **Параметри → Bluetooth та пристрої → Миша**.
2. Доступні параметри:
   - **Основна кнопка миші** — оберіть ліву (за замовчуванням) або праву (для лівшів).
   - **Швидкість вказівника миші** — пересуньте повзунок для збільшення або зменшення швидкості.
   - **Прокручування** — налаштуйте кількість рядків, що прокручуються за один оберт коліщатка.
   - **Прокручувати неактивні вікна при наведенні** — дозволяє прокручувати вміст вікна, не роблячи його активним.
3. Натисніть **"Додаткові параметри миші"** для доступу до розширених налаштувань:
   - Вкладка **"Кнопки"**: швидкість подвійного натискання.
   - Вкладка **"Параметри вказівника"**: точне налаштування швидкості, увімкнення або вимкнення підвищеної точності.
   - Вкладка **"Вказівники"**: зміна зовнішнього вигляду курсору миші (схеми курсорів).
   - Вкладка **"Коліщатко"**: точне налаштування прокручування.

#### Крок 2. Налаштування клавіатури

1. Перейдіть: **Параметри → Час та мова → Ввід → Клавіатура** або **Параметри → Спеціальні можливості → Клавіатура**.
2. Доступні параметри:
   - **Затримка повтору** — час утримання клавіші до початку повторення символу.
   - **Швидкість повтору** — як швидко повторюється символ при утриманні клавіші.
   - **Залипання клавіш** — дозволяє натискати комбінації клавіш послідовно, а не одночасно (корисно для людей з обмеженими можливостями).
   - **Фільтрація натискань** — ігнорування коротких або повторних натискань.
   - **Сенсорна клавіатура** — налаштування екранної клавіатури для сенсорних пристроїв.

### 3.9. Завдання 9. Створення ярликів на робочому столі

**Мета:** Навчитися створювати ярлики для програм, файлів та папок на робочому столі.

#### Спосіб 1. Через контекстне меню робочого столу

1. Натисніть **правою кнопкою миші** на порожньому місці робочого столу.
2. Оберіть **"Створити"** → **"Ярлик"**.
3. У вікні, що відкриється, натисніть **"Огляд..."** та знайдіть потрібну програму або файл.
4. Натисніть **"Далі"**, введіть ім'я ярлика → **"Готово"**.

#### Спосіб 2. Перетягування з меню "Пуск"

1. Відкрийте меню **"Пуск"** → **"Усі додатки"**.
2. Знайдіть потрібну програму.
3. Затисніть ліву кнопку миші та **перетягніть** значок програми на робочий стіл.

#### Спосіб 3. Через Провідник файлів

1. Знайдіть файл або папку у Провіднику файлів.
2. Натисніть **правою кнопкою миші** → **"Показати додаткові параметри"** (у Windows 11) → **"Відправити"** → **"Робочий стіл (створити ярлик)"**.

#### Спосіб 4. Створення ярлика для веб-сайту

1. Натисніть **правою кнопкою миші** на робочому столі → **"Створити"** → **"Ярлик"**.
2. У полі розташування введіть URL-адресу, наприклад: `https://www.google.com`.
3. Натисніть **"Далі"**, введіть назву → **"Готово"**.

#### Відображення системних значків на робочому столі

1. Перейдіть: **Параметри → Персоналізація → Теми → Параметри піктограм робочого столу**.
2. Увімкніть відображення: Комп'ютер, Кошик, Файли користувача, Панель керування, Мережа.
3. Натисніть **"Застосувати"** → **"OK"**.

### 3.10. Завдання 10. Панель швидкого запуску

**Мета:** Навчитися організовувати швидкий доступ до часто використовуваних програм.

> **Примітка:** У Windows 11 класична "Панель швидкого запуску" (Quick Launch) замінена функціональністю панелі задач та меню "Пуск". Нижче описано, як організувати швидкий доступ до програм сучасними засобами.

#### Крок 1. Закріплення програм на панелі задач

1. Знайдіть програму в меню "Пуск" або через пошук.
2. Натисніть правою кнопкою миші → **"Закріпити на панелі задач"**.
3. Значок програми з'явиться на панелі задач для швидкого запуску одним натисканням.

#### Крок 2. Закріплення програм у меню "Пуск"

1. Натисніть правою кнопкою миші на програмі → **"Закріпити на початковому екрані"**.
2. Перетягніть значки для впорядкування.

#### Крок 3. Використання гарячих клавіш для запуску програм з панелі задач

- Програми, закріплені на панелі задач, мають порядкові номери зліва направо.
- Натисніть **Win + 1** для запуску першої закріпленої програми, **Win + 2** — другої і т.д. (до Win + 0 — десятої).
- Якщо програма вже відкрита, ця комбінація перемикає на її вікно.

## 4. Гарячі клавіші Windows

Знання гарячих клавіш значно прискорює роботу з Windows. Запам'ятайте найважливіші комбінації:

### Основні гарячі клавіші

| Комбінація | Дія |
|---|---|
| **Win + D** | Згорнути/відновити всі вікна (показати робочий стіл) |
| **Win + E** | Відкрити Провідник файлів |
| **Win + I** | Відкрити Параметри (Settings) |
| **Win + L** | Заблокувати комп'ютер (екран блокування) |
| **Alt + Tab** | Перемикання між відкритими вікнами |
| **Win + Tab** | Відкрити "Подання задач" (перегляд усіх вікон та віртуальних робочих столів) |
| **Win + S** | Відкрити пошук Windows |
| **Win + A** | Відкрити панель швидких параметрів (Wi-Fi, Bluetooth, гучність, яскравість) |
| **Win + R** | Відкрити вікно "Виконати" (Run) |
| **Win + X** | Відкрити меню швидких посилань (Power User Menu) |

### Знімки екрана

| Комбінація | Дія |
|---|---|
| **Print Screen** | Зробити знімок усього екрана (копіюється в буфер обміну) |
| **Alt + Print Screen** | Зробити знімок активного вікна |
| **Win + Shift + S** | Відкрити інструмент "Ножиці" (Snipping Tool) для вибіркового знімка |
| **Win + Print Screen** | Зробити знімок екрана та автоматично зберегти його у папку "Зображення → Знімки екрана" |

### Керування вікнами

| Комбінація | Дія |
|---|---|
| **Win + стрілка вліво/вправо** | Прикріпити вікно до лівої/правої половини екрана |
| **Win + стрілка вгору** | Розгорнути вікно на весь екран |
| **Win + стрілка вниз** | Згорнути вікно / відновити з повноекранного режиму |
| **Win + Z** | Відкрити макети прикріплення (Snap Layouts) у Windows 11 |
| **Alt + F4** | Закрити активне вікно (на робочому столі — вікно завершення роботи) |

### Віртуальні робочі столи

| Комбінація | Дія |
|---|---|
| **Win + Ctrl + D** | Створити новий віртуальний робочий стіл |
| **Win + Ctrl + стрілка вліво/вправо** | Перемикання між віртуальними робочими столами |
| **Win + Ctrl + F4** | Закрити поточний віртуальний робочий стіл |

### Робота з текстом та файлами

| Комбінація | Дія |
|---|---|
| **Ctrl + C** | Копіювати |
| **Ctrl + X** | Вирізати |
| **Ctrl + V** | Вставити |
| **Ctrl + Z** | Скасувати останню дію |
| **Ctrl + Y** | Повторити скасовану дію |
| **Ctrl + A** | Виділити все |
| **Ctrl + S** | Зберегти |
| **F2** | Перейменувати виділений файл або папку |
| **Delete** | Видалити у Кошик |
| **Shift + Delete** | Видалити безповоротно (без Кошика) |

## 5. Завдання для самостійного опрацювання

1. Встановіть на робочому столі шпалери у режимі **"Слайд-шоу"** з папки "Зображення". Налаштуйте інтервал зміни зображень на 10 хвилин. Опишіть послідовність дій.

2. Створіть **власну тему Windows**: встановіть темний режим, оберіть акцентний колір вручну, увімкніть прозорість, збережіть тему під власним ім'ям. Запишіть назву збереженої теми.

3. Налаштуйте панель задач таким чином: вирівнювання — ліворуч, автоприховування — увімкнено, подання задач — вимкнено, міні-додатки — вимкнено. Зробіть знімок екрана результату (Win + Shift + S).

4. Поясніть, яка **різниця між "Параметрами" (Settings) та "Панеллю керування" (Control Panel)**? У яких ситуаціях краще використовувати кожен з цих інструментів? Наведіть по 2 приклади для кожного.

5. У Диспетчері завдань (Ctrl + Shift + Esc) перейдіть до вкладки "Автозавантаження". Запишіть **3 програми з вашого списку автозапуску**, вказавши для кожної: назву, видавця, стан та рівень впливу на запуск. Які з них можна безпечно вимкнути?

6. Змініть роздільну здатність екрана на **не рекомендовану** (нижчу за поточну), зверніть увагу на зміни в якості зображення. Поверніть рекомендовану роздільну здатність. Опишіть, що відбулося з елементами інтерфейсу.

7. Створіть **три ярлики** на робочому столі різними способами:
   - Ярлик для програми "Блокнот" (Notepad) — через контекстне меню.
   - Ярлик для папки "Документи" — через Провідник файлів.
   - Ярлик для веб-сайту вашого навчального закладу — через поле розташування URL.

8. Додайте **другу мову введення** (наприклад, англійську, якщо вона ще не додана). Перемкніться між мовами за допомогою комбінації Win + Пробіл. Запишіть, які мови введення встановлені на вашому комп'ютері.

9. Перелічіть **щонайменше 10 гарячих клавіш Windows**, які ви запам'ятали з цього уроку, та опишіть дію кожної комбінації своїми словами.

10. Налаштуйте режим **"Не турбувати"** (Do Not Disturb) для сповіщень. Опишіть, у яких ситуаціях цей режим може бути корисним для студента.

11. Використовуючи комбінацію **Win + Z**, прикріпіть два вікна поруч (Snap Layouts): Провідник файлів зліва та Блокнот справа. Зробіть знімок екрана результату.

12. Створіть **новий віртуальний робочий стіл** (Win + Ctrl + D), відкрийте на ньому програму "Калькулятор", потім перемкніться між робочими столами (Win + Ctrl + стрілки). Опишіть, для чого можуть бути корисні віртуальні робочі столи.

13. Змініть **масштабування дисплею** на 125% (або інше значення, відмінне від поточного). Опишіть, як змінився вигляд елементів інтерфейсу. Поверніть початкове значення.

14. Увімкніть відображення **системних значків на робочому столі** (Комп'ютер, Кошик, Мережа). Запишіть шлях до відповідного налаштування.

15. Дослідіть розділ **"Спеціальні можливості"** (Accessibility) у Параметрах Windows. Опишіть 3 функції, які можуть бути корисними для людей з особливими потребами (наприклад, екранна лупа, високий контраст, озвучування).
