## 1. Мета роботи

- Ознайомитися з основними елементами робочого столу Windows.
- Навчитися працювати з меню "Пуск" та панеллю задач.
- Опанувати навігацію у Провіднику Windows (File Explorer).
- Навчитися створювати, копіювати, переміщувати, перейменовувати та видаляти файли й папки.
- Закріпити навички використання гарячих клавіш для роботи з файлами.

---

## 2. Обладнання та програмне забезпечення

- Персональний комп'ютер з операційною системою Windows 10 або Windows 11.
- Клавіатура та миша.

---

## 3. Техніка безпеки

1. Перед початком роботи перевірте справність обладнання.
2. Не вмикайте та не вимикайте комп'ютер без дозволу викладача.
3. Працюйте лише з файлами та папками у вказаних каталогах (робочий стіл або папка вашого облікового запису).
4. Не видаляйте та не змінюйте системні файли.
5. Після завершення роботи приведіть робоче місце до початкового стану.

## 4. Теоретичні відомості

### 4.1. Робочий стіл Windows

**Робочий стіл (Desktop)** — це основний екран, який ви бачите після завантаження Windows. Він є вашим головним робочим простором.

Основні елементи робочого столу:

| Елемент | Опис |
|---------|------|
| **Значки (ярлики)** | Піктограми програм, файлів і папок, розміщені на робочому столі. Подвійний клік відкриває відповідний об'єкт. Стандартні значки: "Цей ПК", "Кошик", "Мережа". |
| **Панель задач** | Горизонтальна смуга внизу екрана. Містить кнопку "Пуск", закріплені програми, відкриті вікна та системний трей. |
| **Кнопка "Пуск"** | Розташована ліворуч (Windows 10) або по центру (Windows 11) панелі задач. Відкриває головне меню операційної системи. |
| **Область сповіщень (системний трей)** | Правий кінець панелі задач. Показує годинник, дату, значки мережі, гучності, стану батареї та інших служб. |
| **Віджети** | Інтерактивні елементи (у Windows 11), які відображають новини, погоду та іншу динамічну інформацію. |

### 4.2. Меню "Пуск"

Меню "Пуск" — це центральний елемент навігації в Windows. Відкривається натисканням кнопки "Пуск" або клавіші **Win** на клавіатурі.

**Структура меню "Пуск" у Windows 11:**

- **Закріплені застосунки (Pinned)** — сітка значків програм, які ви використовуєте найчастіше. Можна додавати та видаляти.
- **Рекомендовані (Recommended)** — нещодавно додані програми та нещодавно відкриті файли.
- **Усі застосунки (All Apps)** — повний алфавітний список усіх встановлених програм (кнопка у верхньому правому куті).
- **Обліковий запис та кнопка живлення** — внизу меню; дозволяє вимкнути, перезавантажити ПК, перейти в режим сну або змінити обліковий запис.
- **Рядок пошуку** — зверху меню; дозволяє шукати програми, файли та налаштування.

### 4.3. Панель задач

**Панель задач (Taskbar)** — горизонтальна смуга внизу екрана, що забезпечує швидкий доступ до програм та керування вікнами.

Основні зони панелі задач:

| Зона | Призначення |
|------|-------------|
| **Кнопка "Пуск"** | Відкриває меню "Пуск" |
| **Поле пошуку / значок пошуку** | Пошук програм, файлів, налаштувань, інформації в інтернеті |
| **Подання задач (Task View)** | Перемикання між відкритими вікнами та віртуальними робочими столами (Win + Tab) |
| **Закріплені програми** | Значки часто використовуваних програм для швидкого запуску |
| **Відкриті програми** | Значки працюючих програм з індикатором внизу |
| **Системний трей** | Значки мережі, гучності, батареї, годинник, центр сповіщень |

**Як налаштувати панель задач:**
- Клік правою кнопкою миші на панелі задач > "Параметри панелі задач".
- Можна: закріпити/відкріпити програми, приховати/показати системні значки, змінити вирівнювання (у Windows 11 — ліворуч або по центру).

### 4.4. Вікна Windows

Кожна програма в Windows працює у вікні. Вікно має стандартні елементи:

| Елемент | Опис |
|---------|------|
| **Рядок заголовка (Title Bar)** | Верхня частина вікна. Містить назву програми або документа. |
| **Кнопка "Згорнути" (Minimize)** | Значок `—` у верхньому правому куті. Згортає вікно на панель задач. |
| **Кнопка "Розгорнути / Відновити" (Maximize / Restore)** | Значок квадрата. Розгортає вікно на весь екран або відновлює попередній розмір. |
| **Кнопка "Закрити" (Close)** | Значок `X`. Закриває вікно та завершує роботу програми. |
| **Смуги прокрутки (Scrollbars)** | З'являються праворуч та/або внизу, якщо вміст не вміщується у вікні. |
| **Рядок стану (Status Bar)** | Нижня частина вікна. Показує додаткову інформацію (кількість об'єктів, розмір тощо). |
| **Межі вікна** | Можна перетягувати для зміни розміру вікна. |

**Керування вікнами гарячими клавішами:**

| Клавіші | Дія |
|---------|-----|
| Alt + F4 | Закрити поточне вікно |
| Win + стрілка вниз | Згорнути / відновити вікно |
| Win + стрілка вгору | Розгорнути вікно на весь екран |
| Win + стрілка ліворуч/праворуч | Прикріпити вікно до лівої/правої половини екрана |
| Alt + Tab | Перемикання між відкритими вікнами |

### 4.5. Провідник Windows (File Explorer)

**Провідник (File Explorer)** — це вбудований файловий менеджер Windows, що дозволяє переглядати, організовувати та керувати файлами й папками на вашому комп'ютері.

**Як відкрити Провідник:**
- Натиснути значок папки на панелі задач.
- Комбінація клавіш **Win + E**.
- Через меню "Пуск" > пошук "Провідник" або "File Explorer".
- Правий клік на кнопці "Пуск" > "Провідник".

**Основні елементи інтерфейсу Провідника:**

| Елемент | Опис |
|---------|------|
| **Панель навігації (Navigation Pane)** | Ліва частина вікна. Відображає дерево папок: "Швидкий доступ", "Цей ПК", "Мережа", диски. |
| **Адресний рядок (Address Bar)** | Зверху. Показує шлях до поточної папки. Можна вводити шлях вручну. |
| **Рядок пошуку (Search Bar)** | У верхньому правому куті. Пошук файлів у поточній папці та вкладених папках. |
| **Панель інструментів / стрічка** | Зверху. Містить кнопки: "Створити", "Вирізати", "Копіювати", "Вставити", "Перейменувати", "Видалити", "Сортування", "Вигляд" тощо. У Windows 11 — спрощена панель команд замість стрічки. |
| **Робоча область** | Центральна частина вікна. Відображає вміст поточної папки (файли та підпапки). |
| **Рядок стану** | Внизу вікна. Показує кількість об'єктів, розмір виділеного файлу тощо. |
| **Вкладки (Tabs)** | У Windows 11 можна відкрити кілька вкладок в одному вікні Провідника (Ctrl + T — нова вкладка). |
| **Кнопки навігації** | "Назад", "Вперед", "Вгору" — для переміщення між папками. |

**Навігація по файловій системі:**
- Подвійний клік на папці — відкриває її.
- Кнопка "Вгору" (або Alt + стрілка вгору) — перехід до батьківської папки.
- Клік на елементі в адресному рядку — перехід до відповідної папки.
- У панелі навігації можна розгортати дерево папок натисканням стрілки зліва від папки.

### 4.6. Операції з файлами та папками

**Створення нової папки:**
- У Провіднику: кнопка "Створити" > "Папка" або комбінація **Ctrl + Shift + N**.
- Правий клік у робочій області > "Створити" > "Папка".

**Створення нового файлу:**
- Правий клік у робочій області > "Створити" > обрати тип файлу (текстовий документ тощо).

**Копіювання файлів/папок:**
1. Виділити об'єкт.
2. **Ctrl + C** (копіювати).
3. Перейти до цільової папки.
4. **Ctrl + V** (вставити).

Альтернатива: правий клік > "Копіювати", потім у цільовій папці правий клік > "Вставити".

**Переміщення файлів/папок:**
1. Виділити об'єкт.
2. **Ctrl + X** (вирізати).
3. Перейти до цільової папки.
4. **Ctrl + V** (вставити).

Альтернатива: перетягнути файл мишею з однієї папки в іншу (на тому ж диску — переміщення, на іншому — копіювання).

**Перейменування:**
- Виділити файл/папку > натиснути **F2** > ввести нове ім'я > **Enter**.
- Або: правий клік > "Перейменувати".

**Видалення:**
- Виділити об'єкт > натиснути **Delete** — переміщення до Кошика.
- **Shift + Delete** — безповоротне видалення (без Кошика). Використовуйте обережно!

**Скасування останньої дії:**
- **Ctrl + Z** — скасовує останню операцію (перейменування, переміщення, видалення тощо).

### 4.7. Кошик

**Кошик (Recycle Bin)** — спеціальна системна папка, куди потрапляють видалені файли та папки.

- Значок Кошика зазвичай знаходиться на робочому столі.
- Видалені файли зберігаються в Кошику, поки ви його не очистите.
- **Відновлення файлів:** відкрийте Кошик > виділіть файл > натисніть "Відновити" (файл повернеться у папку, з якої був видалений).
- **Очищення Кошика:** правий клік на значку Кошика > "Очистити кошик" — видаляє всі файли безповоротно.
- Кошик має обмежений обсяг (зазвичай ~5–10% від обсягу диска). Коли Кошик заповнений, найстаріші файли видаляються автоматично.

### 4.8. Властивості файлів і папок

Щоб переглянути властивості файлу або папки:
- Правий клік на об'єкті > "Властивості" (або виділити об'єкт та натиснути **Alt + Enter**).

Вкладка "Загальні" містить:
- **Тип файлу** та програму, якою він відкривається.
- **Розташування** — повний шлях до файлу.
- **Розмір** та **Розмір на диску**.
- **Дату створення, зміни та останнього доступу**.
- **Атрибути**: "Лише для читання" (захист від змін), "Прихований" (файл не відображається за замовчуванням).

### 4.9. Пошук файлів

У Провіднику є рядок пошуку у верхньому правому куті вікна.

- Введіть назву файлу або його частину — пошук виконується в поточній папці та її підпапках.
- Можна використовувати символи підстановки: `*` (будь-яка кількість символів), `?` (один символ).
- Приклади: `*.txt` — усі текстові файли; `звіт*` — усі файли, що починаються зі слова "звіт".
- Для пошуку по всьому комп'ютеру використовуйте пошук через меню "Пуск" або панель задач.

### 4.10. Гарячі клавіші для роботи з файлами

| Клавіші | Дія |
|---------|-----|
| **Ctrl + C** | Копіювати виділений об'єкт |
| **Ctrl + V** | Вставити |
| **Ctrl + X** | Вирізати (для переміщення) |
| **Ctrl + Z** | Скасувати останню дію |
| **Ctrl + Y** | Повторити скасовану дію |
| **Ctrl + A** | Виділити все |
| **F2** | Перейменувати |
| **Delete** | Видалити в Кошик |
| **Shift + Delete** | Видалити безповоротно |
| **Ctrl + Shift + N** | Створити нову папку |
| **Win + E** | Відкрити Провідник |
| **Alt + Enter** | Властивості виділеного об'єкта |
| **Ctrl + T** | Нова вкладка у Провіднику (Windows 11) |
| **Ctrl + W** | Закрити вкладку / вікно Провідника |
| **Alt + стрілка вгору** | Перейти до батьківської папки |

## 5. Практичні завдання

> **Увага!** Усі завдання виконуйте у папці "Робочий стіл" або "Документи" вашого облікового запису. Не змінюйте системні файли та папки!

---

### Завдання 1. Дослідження робочого столу та панелі задач

1. Розгляньте робочий стіл. Запишіть у зошит, які значки на ньому є.
2. Знайдіть на панелі задач: кнопку "Пуск", поле пошуку, закріплені програми, системний трей (годинник, значок мережі, значок гучності).
3. Наведіть курсор на годинник у системному треї — зверніть увагу на спливаючу підказку з датою.
4. Клікніть правою кнопкою миші на панелі задач > оберіть "Параметри панелі задач". Перегляньте доступні налаштування (не змінюйте їх).

---

### Завдання 2. Робота з меню "Пуск"

1. Натисніть кнопку "Пуск" (або клавішу **Win**).
2. Знайдіть секцію "Закріплені" та секцію "Рекомендовані".
3. Натисніть "Усі застосунки" — перегляньте повний список програм.
4. У рядку пошуку меню "Пуск" введіть "Блокнот" (або "Notepad") — відкрийте знайдену програму.
5. Закрийте Блокнот (кнопка X або **Alt + F4**).

---

### Завдання 3. Робота з вікнами

1. Відкрийте одночасно три програми: Провідник (**Win + E**), Блокнот (через пошук) і Калькулятор (через пошук).
2. Згорніть усі вікна кнопкою "Згорнути" (—) у кожному вікні.
3. Розгорніть Провідник з панелі задач.
4. Розгорніть вікно Провідника на весь екран (кнопка "Розгорнути" або **Win + стрілка вгору**).
5. Відновіть попередній розмір вікна (повторне натискання кнопки або **Win + стрілка вниз**).
6. Прикріпіть Блокнот до лівої половини екрана (**Win + стрілка ліворуч**), а Калькулятор — до правої (**Win + стрілка праворуч**).
7. Перемикайтеся між вікнами за допомогою **Alt + Tab**.
8. Закрийте всі три програми.

---

### Завдання 4. Знайомство з Провідником

1. Відкрийте Провідник (**Win + E**).
2. У панелі навігації зліва знайдіть: "Швидкий доступ" (або "Домашня сторінка"), "Цей ПК", "Робочий стіл", "Документи", "Завантаження".
3. Натисніть "Цей ПК" — перегляньте список дисків (C:, D: тощо).
4. Відкрийте диск C: подвійним кліком. Зверніть увагу на адресний рядок — він показує "Цей ПК > Локальний диск (C:)".
5. Натисніть кнопку "Вгору" (або **Alt + стрілка вгору**) — ви повернетесь до "Цей ПК".
6. Перейдіть до папки "Документи".

---

### Завдання 5. Створення структури папок

1. У папці "Документи" створіть нову папку з вашим прізвищем. Для цього:
   - Натисніть **Ctrl + Shift + N**.
   - Введіть ваше прізвище (наприклад, "Петренко").
   - Натисніть **Enter**.
2. Відкрийте створену папку.
3. Усередині створіть таку структуру підпапок (використовуйте **Ctrl + Shift + N** для кожної):

```
Петренко/
├── Навчання/
│   ├── Конспекти/
│   └── Домашні_завдання/
├── Проєкти/
│   ├── Проєкт_1/
│   └── Проєкт_2/
└── Особисте/
```

4. Переконайтеся, що структура створена правильно, переглянувши дерево папок у панелі навігації.

### Завдання 6. Створення, копіювання та переміщення файлів

1. Відкрийте папку "Конспекти" (всередині "Навчання").
2. Створіть текстовий файл: правий клік > "Створити" > "Текстовий документ". Назвіть його `Урок_01.txt`.
3. Відкрийте файл подвійним кліком, введіть будь-який текст (наприклад, "Це мій перший конспект"), збережіть (**Ctrl + S**) і закрийте.
4. Створіть ще два файли: `Урок_02.txt` та `Урок_03.txt` (можете залишити їх порожніми).
5. **Копіювання:** виділіть файл `Урок_01.txt` > натисніть **Ctrl + C** > перейдіть до папки "Домашні_завдання" > натисніть **Ctrl + V**. Переконайтеся, що файл з'явився в обох папках.
6. **Переміщення:** поверніться до папки "Конспекти" > виділіть `Урок_03.txt` > натисніть **Ctrl + X** > перейдіть до папки "Проєкт_1" > натисніть **Ctrl + V**. Переконайтеся, що файл зник з "Конспекти" та з'явився у "Проєкт_1".

### Завдання 7. Перейменування файлів та папок

1. Перейдіть до папки "Конспекти".
2. Виділіть файл `Урок_02.txt`.
3. Натисніть **F2** — ім'я файлу стане доступним для редагування.
4. Змініть назву на `Операційні_системи_Урок_02.txt` і натисніть **Enter**.
5. Перейдіть до папки "Проєкти".
6. Перейменуйте папку "Проєкт_2" на "Архів" (правий клік > "Перейменувати" або **F2**).

---

### Завдання 8. Зміна вигляду відображення у Провіднику

1. Відкрийте папку "Конспекти".
2. Знайдіть кнопку "Вигляд" (View) на панелі інструментів Провідника.
3. Послідовно переключіть режими відображення та зверніть увагу на відмінності:
   - **Великі значки** — показують великі піктограми файлів.
   - **Дрібні значки** — компактний вигляд.
   - **Список** — файли відображаються у вигляді списку.
   - **Таблиця (Подробиці)** — файли відображаються з додатковою інформацією: ім'я, дата зміни, тип, розмір.
   - **Плитка** — файли з піктограмами середнього розміру та основною інформацією.
4. Залишіть режим "Таблиця" — він є найінформативнішим для роботи з файлами.
5. У режимі "Таблиця" натисніть на заголовок стовпця "Ім'я" — файли відсортуються за алфавітом. Натисніть ще раз — сортування зміниться на зворотне (від Я до А).
6. Спробуйте відсортувати за "Датою зміни" та за "Розміром".

---

### Завдання 9. Видалення файлів та робота з Кошиком

1. Перейдіть до папки "Проєкт_1".
2. Виділіть файл `Урок_03.txt`.
3. Натисніть клавішу **Delete**. Підтвердіть видалення, якщо з'явиться запит.
4. Відкрийте **Кошик** на робочому столі (подвійний клік на значку Кошика).
5. Знайдіть у Кошику файл `Урок_03.txt`.
6. Виділіть його > натисніть "Відновити" (або правий клік > "Відновити"). Файл повернеться у папку "Проєкт_1".
7. Переконайтеся, що файл знову з'явився у папці "Проєкт_1".
8. Тепер виділіть цей файл знову та натисніть **Shift + Delete**. У вікні підтвердження натисніть "Так" — файл буде видалено безповоротно (не потрапить до Кошика).

### Завдання 10. Перегляд властивостей та пошук файлів

**Частина А — Властивості:**
1. Перейдіть до папки "Конспекти".
2. Виділіть файл `Урок_01.txt`.
3. Натисніть **Alt + Enter** (або правий клік > "Властивості").
4. У вікні "Властивості" знайдіть та запишіть у зошит:
   - Тип файлу.
   - Розмір файлу.
   - Розташування (повний шлях).
   - Дату створення.
5. Закрийте вікно властивостей.
6. Перегляньте властивості папки "Навчання" — зверніть увагу на загальний розмір папки та кількість файлів і вкладених папок.

**Частина Б — Пошук:**
1. Відкрийте Провідник і перейдіть до папки з вашим прізвищем.
2. У рядку пошуку (верхній правий кут) введіть `Урок` і натисніть **Enter**.
3. Дочекайтеся результатів — мають знайтися всі файли, назва яких містить "Урок", у всіх підпапках.
4. Спробуйте пошук `*.txt` — мають знайтися всі текстові файли.
5. Запишіть у зошит, скільки файлів було знайдено кожним запитом.

## 7. Завдання для самостійного опрацювання

1. Створіть на робочому столі папку "Тест_СР" та всередині неї — три вкладені папки: "Альфа", "Бета", "Гамма". У кожній з цих папок створіть по два текстових файли з довільними назвами. Опишіть покроково, як ви це зробили.

2. Поясніть, що відбудеться, якщо перетягнути файл мишею з папки на диску C: до папки на диску D:. А що відбудеться, якщо перетягнути файл між двома папками на одному диску? Чому результат різний?

3. Дослідіть, як виділити кілька файлів одночасно. Випробуйте три способи: за допомогою Ctrl + клік (окремі файли), Shift + клік (діапазон файлів) та Ctrl + A (усі файли). Опишіть різницю між цими способами.

4. Знайдіть на комп'ютері всі файли з розширенням `.log`, використовуючи пошук у Провіднику від кореня диска C:. Скільки файлів знайдено? Де вони розташовані?

5. У вікні властивостей будь-якого файлу встановіть атрибут "Лише для читання". Потім спробуйте відредагувати цей файл у Блокноті та зберегти зміни. Що відбулося? Зніміть атрибут після завершення експерименту.

6. Дослідіть, як працює комбінація клавіш **Ctrl + Z** у Провіднику. Виконайте три послідовні дії (наприклад, перейменування, переміщення, копіювання), а потім скасуйте їх по одній за допомогою Ctrl + Z. Скільки дій вдалося скасувати?

7. Дізнайтеся, як налаштувати розмір Кошика. Знайдіть у властивостях Кошика, який максимальний обсяг він може зайняти на вашому диску. Як змінити цей обсяг?

8. Використовуючи адресний рядок Провідника, перейдіть безпосередньо до папки `C:\Windows\Fonts`. Що зберігається в цій папці? Скільки там об'єктів? Яке розширення мають файли шрифтів?

9. Дослідіть, як створити ярлик для файлу або папки. Створіть ярлик для папки "Документи" на робочому столі. Чим ярлик відрізняється від самого файлу або папки? Що відбудеться, якщо видалити ярлик — чи видалиться оригінальний об'єкт?

10. Знайдіть та опишіть щонайменше п'ять гарячих клавіш Windows, які НЕ були згадані в цій практичній роботі. Поясніть, для чого кожна з них використовується.

11. Дослідіть контекстне меню (правий клік) для файлу та для папки. Чим відрізняються пункти меню для файлу та для папки? Назвіть щонайменше три відмінності.

12. Створіть у папці "Документи" файл із назвою, що містить спеціальні символи (наприклад, крапки, пробіли, дужки). Які символи Windows дозволяє використовувати в іменах файлів, а які — забороняє? Спробуйте створити файл із символами `\ / : * ? " < > |` та опишіть результат.
